# 文献库说明

本仓库用于维护组内论文资料，文档清单分为两部分：

- 论文列表：老师发送或指定阅读的论文。
- ICML读书小组：组会论文阅读的文章。

## 使用约定

- 日期格式统一为 `YYYY-MM-DD`。
- “论文列表”的日期建议填写老师发送或加入文献库的日期。
- “ICML读书小组”的日期建议填写组会汇报日期。
- “论文”列放项目内相对链接，链接到 `论文库` 中对应 PDF 或资料文件。
- 建议论文文件按如下目录存放：
  - `论文库/论文列表/`
  - `论文库/ICML读书小组/`

## 链接写法

Markdown 链接格式为：

```md
[显示文字](文件路径)
```

例如：

```md
[PDF](论文库/论文列表/Paper_Title_2026.pdf)
```

如果文件名中有空格，建议在链接中把空格写成 `%20`，例如：

```md
[PDF](论文库/论文列表/A%20Decade%20of%20Deep%20Learning-based%20Biomedical%20Image%20Segmentation_bioRxiv_2026.pdf)
```

## 维护步骤

新增论文时，建议同步完成三步：

1. 将论文 PDF 放入对应目录。
2. 将文件名改为清晰格式，例如 `Paper_Title_2026.pdf`。
3. 在 [文档清单](文档清单.md) 的对应表格新增一行，并把“论文”列链接到该文件。
